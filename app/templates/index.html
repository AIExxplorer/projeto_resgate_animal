Bem-vindo ao Sistema de Gestão de Resgate Animal

Este sistema foi desenvolvido para auxiliar na gestão de uma instituição dedicada ao resgate de animais.

Aqui você pode:

  • Gerenciar informações sobre os animais resgatados
  • Controlar doações recebidas
  • Manter um cadastro de doadores

Para acessar as funcionalidades completas, por favor, utilize a interface gráfica do sistema.
